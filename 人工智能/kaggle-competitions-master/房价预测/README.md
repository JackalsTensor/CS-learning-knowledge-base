# Kaggle 学生成绩预测项目

## 项目简介
本项目基于 Kaggle 学生成绩预测竞赛数据集，通过机器学习模型预测学生考试分数，分析影响成绩的关键因素，为教育数据分析提供参考。

## 数据集
- **训练集**：`训练集.csv`，包含学生基本信息、学习行为、生活习惯等特征及考试分数标签
- **测试集**：`测试集.csv`，用于模型最终预测与赛事提交
- **数据来源**：Kaggle 学生成绩预测竞赛


## 核心对比结论
根据 Kaggle 官方评分结果，**XGBoost 模型表现显著更优**：

- 私人评分从 `8.89192` → `8.76224`，误差降低约 **1.46%**
- 公开评分从 `8.87318` → `8.73460`，误差降低约 **1.56%**

说明 XGBoost 能更好地捕捉学生学习行为、生活习惯与考试分数之间的复杂非线性关系，泛化能力更强。
