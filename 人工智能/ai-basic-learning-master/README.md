# AI-Basic-Learning
人工智能基础学习仓库 

## 仓库介绍
本仓库用于个人人工智能方向的日常学习记录以及机器学习实战项目的整理，所有项目均为边学边做、自主编写、持续迭代，包含代码实现、项目案例、数据处理练习等内容。

## 包含内容
1. 机器学习，深度学习等经典入门项目（鸢尾花分类、数据分析、回归/分类任务）
2. 常用算法实践（KNN、决策树、逻辑回归等）
3. 数据集处理、数据可视化、模型评估与训练代码.

## 项目特点
- 全部代码为自主学习编写，持续更新、不断完善
- 结构清晰、分类规范，方便回顾与展示

## 使用说明
所有代码均基于 Python 实现，依赖常用机器学习等第三方库
