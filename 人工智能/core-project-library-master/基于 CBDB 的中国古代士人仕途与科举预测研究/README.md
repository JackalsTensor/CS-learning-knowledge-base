# 基于CBDB的中国古代士人仕途与科举预测研究

## 项目简介
本项目依托中国历代人物传记数据库（CBDB），结合机器学习方法，量化分析古代士人科举成功与仕途发展规律，并重点挖掘方氏家族的科举优势及核心影响因素，为历史家族与科举制度研究提供数据驱动的新范式。

## 研究背景
中国古代科举制度是选拔官员的核心机制，深刻影响士人阶层的社会流动与家族兴衰。本研究在传统史学定性分析的基础上，引入机器学习方法，实现从“经验描述”到“规律预测”的跨越。

## 核心研究问题
1.  基于人物籍贯、家族、生卒时期、社会关系等特征，**预测古代士人科举是否成功**；
2.  对科举成功者，进一步**预测其最终官职高低**，量化仕途发展潜力；
3.  以方氏家族为典型案例，挖掘其在科举与仕途上的**相对优势及关键影响因素**。

## 研究方法
- **数据预处理**：清洗CBDB原始数据，提取科举、官职、家族、地域、时间等核心字段，构建建模数据集；
- **标签构建**：将“是否考中进士/举人”作为科举成功标签，将“最高官职等级”作为仕途表现标签；
- **特征工程**：从个人属性、家族传承、地域环境、时代背景四个维度构建预测特征；
- **模型训练**：采用随机森林、XGBoost等模型，分别训练科举成功分类器与官职等级预测器；
- **案例分析**：聚焦方氏家族，对比其与整体样本的表现差异，通过特征重要性分析识别家族优势来源。

## 预期成果
- 构建一套可复现的古代士人仕途预测流程，实现较高精度的科举与官职预测；
- 量化揭示方氏家族在科举竞争中的核心优势（如地域聚集、家族传承、时代机遇等）；
- 为历史社会学、家族史研究提供量化分析的新工具与实证结论。

## 技术栈
- 数据处理：`pandas`、`numpy`
- 可视化：`matplotlib`
- 机器学习：`scikit-learn`、`xgboost`
