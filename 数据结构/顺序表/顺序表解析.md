# 顺序表（Sequential List）

## 一、什么是顺序表

顺序表（Sequential List）是一种使用**连续存储空间**保存数据元素的线性表。

本质上就是利用数组来实现线性表。

例如：

```text
下标: 0   1   2   3   4
数据: 10  20  30  40  50
```

在内存中连续存放：

```text
┌────┬────┬────┬────┬────┐
│ 10 │ 20 │ 30 │ 40 │ 50 │
└────┴────┴────┴────┴────┘
```

---

## 二、顺序表结构定义

```c
#define MAXSIZE 100

typedef struct
{
    int data[MAXSIZE];
    int length;
} SeqList;
```

### 成员说明

| 成员     | 作用     |
| ------ | ------ |
| data[] | 存放数据   |
| length | 当前元素个数 |

---

## 三、基本操作

### 1. 初始化

```c
void InitList(SeqList *L)
{
    L->length = 0;
}
```

时间复杂度：

```text
O(1)
```

---

### 2. 插入元素

```c
int ListInsert(SeqList *L,int pos,int e)
{
    if(pos<1||pos>L->length+1)
        return 0;

    for(int i=L->length;i>=pos;i--)
        L->data[i]=L->data[i-1];

    L->data[pos-1]=e;
    L->length++;

    return 1;
}
```

示例：

```text
原表：
10 20 30 40

在第2个位置插入99

结果：
10 99 20 30 40
```

时间复杂度：

```text
O(n)
```

---

### 3. 删除元素

```c
int ListDelete(SeqList *L,int pos,int *e)
{
    if(pos<1||pos>L->length)
        return 0;

    *e=L->data[pos-1];

    for(int i=pos;i<L->length;i++)
        L->data[i-1]=L->data[i];

    L->length--;

    return 1;
}
```

示例：

```text
原表：
10 20 30 40

删除第2个元素

结果：
10 30 40
```

时间复杂度：

```text
O(n)
```

---

### 4. 按位查找

```c
int GetElem(SeqList L,int pos)
{
    return L.data[pos-1];
}
```

时间复杂度：

```text
O(1)
```

---

### 5. 按值查找

```c
int LocateElem(SeqList L,int e)
{
    for(int i=0;i<L.length;i++)
    {
        if(L.data[i]==e)
            return i+1;
    }

    return 0;
}
```

时间复杂度：

```text
O(n)
```

| 删除   | O(n)  |

## 五、顺序表优缺点

### 优点

* 支持随机访问
* 查询速度快
* 存储结构简单
* 实现容易

### 缺点

* 插入删除效率低
* 必须连续存储空间
* 容量固定

---

## 六、顺序表与链表对比

| 项目    | 顺序表  | 链表   |
| ----- | ---- | ---- |
| 存储方式  | 连续   | 非连续  |
| 查找    | O(1) | O(n) |
| 插入    | O(n) | O(1) |
| 删除    | O(n) | O(1) |
| 空间利用率 | 较低   | 较高   |

## 九、总结

顺序表是最基础的线性结构，其底层依赖数组实现。

核心特点：

✔ 连续存储

✔ 支持随机访问

✔ 查询快

✔ 插入删除慢

学习数据结构时，应首先掌握顺序表，再学习链表、栈、队列等高级结构。
