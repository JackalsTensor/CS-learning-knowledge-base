# 单链表（Singly Linked List）

## 一、什么是单链表

单链表（Single Linked List）是一种链式存储结构。

每个结点由两部分组成：

```text
┌───────┬────────┐
│ data  │ next   │
└───────┴────────┘
```

* data：存储数据
* next：指向下一个结点

多个结点通过指针连接形成线性结构。

示例：

```text
Head
 ↓
┌───┬───┐    ┌───┬───┐    ┌───┬─────┐
│10 │ ●─┼──► │20 │ ●─┼──► │30 │NULL │
└───┴───┘    └───┴───┘    └───┴─────┘
```

---

# 二、链表结点定义

```c
typedef struct Node
{
    int data;
    struct Node *next;
}Node;
```

---

# 三、头结点

通常会额外设置一个头结点：

```text
Head
 ↓
┌─────┬───┐
│ NULL│ ●─┼──► 第一个数据结点
└─────┴───┘
```

作用：

* 统一插入删除操作
* 方便算法实现
* 避免特殊情况讨论

# 四、基本操作

## 1. 初始化链表

```c
Node *InitList()
{
    Node *head=(Node *)malloc(sizeof(Node));
    head->next=NULL;
    return head;
}
```

时间复杂度：

```text
O(1)
```

---

## 2. 头插法

```c
void HeadInsert(Node *head,int x)
{
    Node *s=(Node *)malloc(sizeof(Node));

    s->data=x;

    s->next=head->next;
    head->next=s;
}
```

示例：

```text
插入前：

Head → 10 → 20 → 30

头插40：

Head → 40 → 10 → 20 → 30
```

时间复杂度：

```text
O(1)
```

---

## 3. 尾插法

```c
void TailInsert(Node *head,int x)
{
    Node *p=head;

    while(p->next)
        p=p->next;

    Node *s=(Node *)malloc(sizeof(Node));

    s->data=x;
    s->next=NULL;

    p->next=s;
}
```

示例：

```text
Head → 10 → 20 → 30

尾插40

Head → 10 → 20 → 30 → 40
```

时间复杂度：

```text
O(n)
```

---

## 4. 按位查找

```c
Node *GetElem(Node *head,int pos)
{
    Node *p=head->next;
    int j=1;

    while(p&&j<pos)
    {
        p=p->next;
        j++;
    }

    return p;
}
```

时间复杂度：

```text
O(n)
```

---

## 5. 按值查找

```c
Node *LocateElem(Node *head,int x)
{
    Node *p=head->next;

    while(p)
    {
        if(p->data==x)
            return p;

        p=p->next;
    }

    return NULL;
}
```

时间复杂度：

```text
O(n)
```

---

## 6. 插入结点

```c
int ListInsert(Node *head,int pos,int x)
{
    Node *p=head;

    int j=0;

    while(p&&j<pos-1)
    {
        p=p->next;
        j++;
    }

    if(!p)
        return 0;

    Node *s=(Node *)malloc(sizeof(Node));

    s->data=x;
    s->next=p->next;
    p->next=s;

    return 1;
}
```

时间复杂度：

```text
O(n)
```

---

## 7. 删除结点

```c
int ListDelete(Node *head,int pos)
{
    Node *p=head;

    int j=0;

    while(p->next&&j<pos-1)
    {
        p=p->next;
        j++;
    }

    if(!p->next)
        return 0;

    Node *q=p->next;

    p->next=q->next;

    free(q);

    return 1;
}
```

时间复杂度：

```text
O(n)
```

# 五、常见算法

## 求链表长度

```c
int Length(Node *head)
{
    int len=0;

    Node *p=head->next;

    while(p)
    {
        len++;
        p=p->next;
    }

    return len;
}
```

---

## 反转链表

```c
Node *Reverse(Node *head)
{
    Node *pre=NULL;
    Node *cur=head->next;

    while(cur)
    {
        Node *next=cur->next;

        cur->next=pre;

        pre=cur;
        cur=next;
    }

    head->next=pre;

    return head;
}
```

---

## 查找中间结点

快慢指针：

```c
Node *MiddleNode(Node *head)
{
    Node *slow=head->next;
    Node *fast=head->next;

    while(fast&&fast->next)
    {
        slow=slow->next;
        fast=fast->next->next;
    }

    return slow;
}
```

# 六、顺序表与单链表比较

| 项目    | 顺序表  | 单链表  |
| ----- | ---- | ---- |
| 存储方式  | 连续存储 | 链式存储 |
| 随机访问  | O(1) | O(n) |
| 插入    | O(n) | O(1) |
| 删除    | O(n) | O(1) |
| 空间利用率 | 较低   | 较高   |

# 九、总结

单链表是最基础的链式存储结构。

核心特点：

✔ 动态存储

✔ 插入删除效率高

✔ 不支持随机访问

✔ 查找效率较低

掌握单链表后，可以进一步学习：

* 双链表
* 循环链表
* 栈
* 队列
* 树
* 图

它是数据结构学习的重要基础。
