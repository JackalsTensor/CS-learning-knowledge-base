# 双链表（Doubly Linked List）

## 一、什么是双链表

双链表（Doubly Linked List）是在单链表基础上的扩展。

单链表每个结点只有一个后继指针：

```text
[data | next]
```

双链表每个结点拥有两个指针：

```text
[prior | data | next]
```

其中：

* prior：指向前驱结点
* data：存储数据
* next：指向后继结点

结构示意图：

```text
NULL ← [10] ⇄ [20] ⇄ [30] → NULL
```

双链表支持：

* 从前向后遍历
* 从后向前遍历

---

# 二、双链表结点定义

```c
typedef struct DNode
{
    int data;
    struct DNode *prior;
    struct DNode *next;
}DNode;
```

---

# 三、头结点

通常采用带头结点的双链表。

```text
Head
 ↓
┌─────┬───────┬──────┐
│NULL │ prior │ next │
└─────┴───────┴───┬──┘
                  │
                  ▼
            第一个数据结点
```

优点：

* 插入方便
* 删除方便
* 减少特殊情况处理

# 四、基本操作

## 1. 初始化

```c
DNode *InitList()
{
    DNode *head=(DNode *)malloc(sizeof(DNode));

    head->prior=NULL;
    head->next=NULL;

    return head;
}
```

时间复杂度：

```text
O(1)
```

---

## 2. 尾插法建立双链表

```c
void TailInsert(DNode *head,int x)
{
    DNode *p=head;

    while(p->next)
        p=p->next;

    DNode *s=(DNode *)malloc(sizeof(DNode));

    s->data=x;
    s->next=NULL;
    s->prior=p;

    p->next=s;
}
```

示例：

```text
Head → 10 → 20 → 30

尾插40

Head → 10 → 20 → 30 → 40
```

时间复杂度：

```text
O(n)
```

---

## 3. 插入结点

在第 pos 个位置插入元素 x。

```c
int ListInsert(DNode *head,int pos,int x)
{
    DNode *p=head;

    int j=0;

    while(p&&j<pos-1)
    {
        p=p->next;
        j++;
    }

    if(!p)
        return 0;

    DNode *s=(DNode *)malloc(sizeof(DNode));

    s->data=x;

    s->next=p->next;
    s->prior=p;

    if(p->next)
        p->next->prior=s;

    p->next=s;

    return 1;
}
```

示例：

```text
原链表：

10 ⇄ 20 ⇄ 30

在第2个位置插入99

结果：

10 ⇄ 99 ⇄ 20 ⇄ 30
```

时间复杂度：

```text
O(n)
```

---

## 4. 删除结点

```c
int ListDelete(DNode *head,int pos)
{
    DNode *p=head->next;

    int j=1;

    while(p&&j<pos)
    {
        p=p->next;
        j++;
    }

    if(!p)
        return 0;

    p->prior->next=p->next;

    if(p->next)
        p->next->prior=p->prior;

    free(p);

    return 1;
}
```

示例：

```text
原链表：

10 ⇄ 20 ⇄ 30

删除20

结果：

10 ⇄ 30
```

时间复杂度：

```text
O(n)
```

---

## 5. 正向遍历

```c
void PrintForward(DNode *head)
{
    DNode *p=head->next;

    while(p)
    {
        printf("%d ",p->data);
        p=p->next;
    }
}
```

---

## 6. 逆向遍历

```c
void PrintBackward(DNode *head)
{
    DNode *p=head;

    while(p->next)
        p=p->next;

    while(p!=head)
    {
        printf("%d ",p->data);
        p=p->prior;
    }
}
```

# 五、双链表核心操作图解

## 插入结点

插入前：

```text
A ⇄ B
```

插入新结点 S：

```text
A ⇄ S ⇄ B
```

操作步骤：

```c
s->next = p->next;
s->prior = p;

if(p->next)
    p->next->prior = s;

p->next = s;
```

---

## 删除结点

删除前：

```text
A ⇄ B ⇄ C
```

删除 B：

```text
A ⇄ C
```

操作步骤：

```c
p->prior->next = p->next;

if(p->next)
    p->next->prior = p->prior;
```

# 七、单链表与双链表对比

| 项目   | 单链表  | 双链表  |
| ---- | ---- | ---- |
| 指针数  | 1个   | 2个   |
| 空间开销 | 小    | 大    |
| 查找前驱 | O(n) | O(1) |
| 正向遍历 | 支持   | 支持   |
| 逆向遍历 | 不支持  | 支持   |
| 插入删除 | 较方便  | 更方便  |

# 十一、总结

双链表是单链表的升级版本。

核心特点：

✔ 支持双向遍历

✔ 插入删除更加方便

✔ 能快速找到前驱结点

✔ 空间开销更大

学习双链表后，可以进一步学习：

* 循环链表
* 栈
* 队列
* 二叉树
* 图

双链表也是理解 LRU 缓存、浏览器历史记录、操作系统页面置换算法的重要基础。
